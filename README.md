
# STM32F103C8T6_FOC
STM32F103C8T6_FOC is an open-source project designed for controlling NEMA 17 stepper motors using Field Oriented Control (FOC). It is based on the STM32F103C8T6 microcontroller and provides a reference PCB for building a precise and smooth stepper motor controller. The project is aimed at hobbyists and developers who want to drive NEMA 17 motors efficiently for applications like CNC machines, 3D printers, or robotic systems.
